Circuit Weaver 基于开源项目 **[Quirk](http://algassert.com/quirk)** 定制, 其作者为 Craig Gidney.

我简化了其功能，便于初学者学习量子线路的开发。
详细培训文档链接： **[Guides](http://qubits.top/guides.html)** 


### 如何构建项目

```bash
npm install
npm run build
```

构建成功后，生成文件如下：
 `out/CircuitWeaver.html`
